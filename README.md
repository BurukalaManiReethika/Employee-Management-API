# Employee-Management-API
# Employee Management System (ASP.NET Web API)

## 📌 Overview
This project is a RESTful API built using ASP.NET Web API and C# to manage employee data.

## 🚀 Features
- CRUD operations (Create, Read, Update, Delete)
- SQL Server database integration
- REST API architecture
- Error handling and validation
- Swagger API documentation

## 🛠 Tech Stack
- ASP.NET Web API
- C#
- Entity Framework Core
- SQL Server
- Postman

## 🔧 How to Run
1. Clone the repository
2. Open in Visual Studio
3. Run migrations:
   - Add-Migration InitialCreate
   - Update-Database
4. Run the project

## 📡 API Endpoints
- GET /api/employee
- POST /api/employee
- PUT /api/employee/{id}
- DELETE /api/employee/{id}

## 📷 Testing
Tested using Postman and Swagger UI

## 📈 Outcome
Improved understanding of backend development, API design, and debugging.
